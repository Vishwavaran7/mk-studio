> **MK Studio** is a premium, highly stylized frontend web experience designed for an editorial wedding and portrait photography studio. Emphasizing luxury minimalism, it pairs dynamic custom animations with high-definition visual layouts to showcase portfolio work.

---

## 🌟 Key Features

*   📸 **Editorial Photography Portfolio:** Responsive masonry and grid layouts showcasing fine art wedding, editorial, and portrait photography.
*   🎭 **Luxury Visual Micro-Interactions:**
    *   **Ken Burns Zoom Header:** A panning hero zoom effect (`.ken-burns` keyframes) creating a cinematic introduction.
    *   **Fade-in Animations:** Smooth scroll transitions (`.fade-in`) showing cards and sections elegantly as the user scrolls.
    *   **Elegant Hover States:** Dynamic image scaling and caption reveals on portfolio items.
*   ✨ **Tailwind Custom Configuration:** Loaded via CDN with extended styling variables:
    *   **Fonts:** Cormorant Garamond (Editorial Serif) & Montserrat (Modern Sans-Serif).
    *   **Colors:** Luxury palette featuring Cream (`#FDFBF7`), Beige (`#F5EFE6`), Gold (`#C5A880`), Charcoal (`#1A1A1A`), and Accent brown (`#8C7A6B`).
    *   **Scrollbars:** Custom gold-accented scrollbars mapping to the studio's branding guidelines.
*   📱 **Fully Responsive Layout:** Optimized mobile/desktop grid structures ensuring premium look on iPhones, tablets, and wide screens.

## 🚀 Local Setup & Running

### 1. Clone the Repository
```bash
git clone https://github.com/Vishwavaran7/mk-studio.git
cd mk-studio
```

### 2. Launching Locally
Open `index.html` in any web browser to view the application, or start a local server:
```bash
# Python local server
python -m http.server 8000
```
Then visit `http://localhost:8000` in your web browser.
